| SQL 절    | 사용 목적                            | 예시                                                                                                                                     |
| -------- | -------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------- |
| SELECT   | 계산된 결과를 출력하거나 새로운 열을 만들 때 사용     | `SELECT ProductName, ROUND(Price * 1.1, 0) AS TaxedPrice FROM Products`                                                                |
| WHERE    | 특정 조건에 따라 행을 필터링할 때 사용           | `SELECT * FROM Orders WHERE ABS(Quantity - Target) > 5`                                                                                |
| HAVING   | GROUP BY 로 그룹화된 결과에 조건을 적용할 때 사용 | `SELECT Dept, SUM(Sales) FROM Employee GROUP BY Dept HAVING SUM(Sales) > 1000` <br>(이 경우 SUM 이 집계 함수지만, 여기에 ROUND 같은 수학 함수를 적용할 수도 있음) |
| ORDER BY | 계산된 값에 따라 결과를 정렬할 때 사용           | `SELECT Name, Salary FROM Employees ORDER BY CEIL(Salary / 1000)` -> (천 단위로 올림하여 정렬)                                                   |
| UPDATE   | 기존 열의 값을 수정할 때 사용                | `UPDATE Products SET Price = ROUND(Price * 1.05, 2)`                                                                                   |

 - 거의 모든 절에서 사용 가능 
 - Mod
	 - 나머지 값을 구할 때 사용
	 - A 값이 작아 나머지 값을 구할 수 없으면 A 값 그대로 출력
	 - `Select mod(3,2) from dual`
		 - Mod(a, b) 중 b 를 a 미만이 될 때 까지 최대로 곱하고 남은 값을 출력
		 - 2 x ? = 3미만 // 2 x 1 = 2 = b // a - b // 3 - 2 = 1 // 출력 값 1
 - Abs
	 - 절대값을 구할 때 사용
	 - `Select abs(n) > abs(-5) => 5`
 - Round
	 - 자릿수를 지정하고 반올림하여 출력
		 - 자릿수 지정 없을 시 정수로 반올림
		 - `Round(8.3438,3)` > 출력 값 8.344
		 - `Round(1165.3438,-2)` > 출력 값 1200
 - Ceil
	 - 올림함수 : 소수 0.1을 정수1로 올려서 출력
		 - `Ceil (7.2)` > 출력 값 8
 - Floor
	 - 내림함수 : 소수 0.9을 정수 0으로 내려서 출력
		 - `Floor (6.8)` > 출력 값 6
	 - 산술연산자를 이용하여 반내림 출력이 가능
 - Power
	 - 제곱 : 복리 이자 계산(지수), 기하학 공식(피타고라스 정리), 데이터 용량(MB, GB), 알고리즘 효율성 분석 등
	 - A 의 B 제곱 값을 반환
	 - POWER(A, B) / POW(A, B) > `POWER(2, 3)` => 8
 - SQRT
	 - 제곱근 : 면적, 거리 계산 (피타고라스 정리), 물리학, 공학, 통계학 등에서 활용 (통계)
	 - N 의 제곱근 값을 반환
	 - SQRT(N) > `SQRT(9)` => 3
