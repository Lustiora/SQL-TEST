 - **Order By 절**^orderby
	 - 정렬을 위해 사용
	 - 출력되는 Column 순서에 따라 1 , 2 , 3 , 4 ~ 등을 Column Name 대신 사용 가능
	 - Select 절에서 별칭 (as) 으로 지정된 Column 이 있다면 별칭만 작성하여도 정렬 가능
	 - Asc (기본) -> 오름차순 정렬 1 2 3 4
	 - Desc -> 내림차순 정렬 4 3 2 1

<p></p>

 - **OFFSET ... FETCH NEXT 절**^offset
	 - 출력 행의 개수를 지정하고 건너뛰고 출력 가능
	 - Order by 절 이후에 작성
		 - `Limit 5`
			 - `OFFSET 0 ROWS FETCH NEXT 5 ROWS ONLY` 0s ~ 5e
		 - `Limit 5, 10`
			 - `OFFSET 5 ROWS FETCH NEXT 5 ROWS ONLY` 5s ~ 5e
			 - 5번 행 이후로 5개 행을 표시

<p></p>

 - **Limit 절**^limit
	 - MySQL, PostgreSQL 에서 사용되는 비표준 ( Offset 절 표준 )
	 - 출력할 행의 개수를 지정 가능
	 - Order by 절 이후에 작성
		 - `Limit 10`
			 - 출력 결과의 행을 10개까지 표시 <= 10
		 - `Limit 5, 10`
			 - 출력 결과의 행을 6부터 10까지 표시 5 < | <= 10
