 - To_char
	 - 형식 변환을 위해 사용 (날짜, 시간, 숫자를 문자열로)
	 - `To_char(Date Column, 'yyyy-mm-dd hh:mi:ss')`
		 - Date Column 을 지정한 형식으로 변환 => 2025-11-27 12:03:38
	 - `To_char(Number column, '999,999,999')`
		 - Number column 에 지정한 형식만큼 콤마 규칙을 적용하여 출력
	 - Format (MySQL)
		 - `Format(Date column, 0)` / 10000000.0 => 10,000,000
			 - 값에 세 자리마다 쉼표(,)를 붙여주고, 소수점은 0자리까지만 표시 (반올림)
			 - Round 와 비슷하지만 문자열 (VARCHAR)로 반환
		 - `Date_format(Date column, '%Y-%m-%d')` => 2025-12-12
			 - [포맷](#^dateformat) 참고
 - Substring
	 - 문자열의 일부만 출력하기 위해 사용하는 구문
	 - `SUBSTRING(Column, 시작 위치, 종료 위치)`
	 - 기본 시작 위치 1
 - Length
	 - 문자열 개수를 확인하기 위해 사용
	 - Substring 의 시작 위치, 종료 위치로 사용 가능
 - Concat / ||
	 - 문자열 추가를 위해 사용
	 - `Concat(DATE Column, 'cm')`
		 - DATE Column 값 뒤에 'cm'을 붙여서 출력
	 - 앞, 뒤, 추가되는 문자열, Column 개수 제한 없음
	 - `DATE Column || 'cm'`
		 - Concat 보다 편리하게 문자열 추가 가능
 - CAST
	 - 문자열 데이터 형식 변환에 사용
	 - 문자열 bytes 한계를 올릴 때도 사용 가능
	 - `Cast(DATE Column as [데이터 형식])`
	 - `cast(f.rental_duration||'days' as interval)`
		 - f.rental_duration Column 타입을 interval로 변환하여 7 days 로 출력
		 - `f.rental_duration * INTERVAL '1 day'`
			® 동일한 출력 , 더욱 나은 가독성 및 편의성
 - String_AGG
	 - Column의 행 값을 열 값으로 변환
	 - 변환되는 값의 중복을 제거하고 값 사이에 문자 추가 가능
	 - Order By를 사용하여 열의 정렬상태 조정 가능
		 - 미사용시 행 값 순서대로 나열
	 - `STRING_AGG(DISTINCT a2.territory_reach, ' , ' ORDER BY a2.territory_reach)`
