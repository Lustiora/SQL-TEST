 - Select 절, Having 절에 사용
 - 함수 내부에 distinct 를 사용하여 중복 값을 제거한 상태로 집계 가능
 - 집계함수(집계함수()) OVER() 형태의 중첩활용 가능
 - Sum
	 - 결과값을 합산할 때 사용
	 - 계산해야 할 값에 Null 이 포함되어 있으면 제외하고 계산
	 - `Select sum(sal) from emp where job='SALESMAN'`
		 - Emp Table 의 job Column 에서 'SALESMAN' 에 해당하는 sal Column 행 값을 합산하여 출력
 - Max / Min
	 - 기본적으로 최대값 / 최소값을 출력할 때 사용
	 - Case문에 사용하여 집계함수 조건으로 사용하고 Group By를 적용하여 Group 내 특정 조건을 만족하는 항목만 출력
	 - `Select max(sal) from emp where job='MANAGER'`
		 - Emp Table 의 job Column 에서 'MANAGER' 에 해당하는 sal Column 행 값 중 `[최대값]`을 출력
		 - Min 을 사용하면 최소값을 출력
	 - Max : 숫자 집계 함수
		 - 최대값을 구할 때 사용되기도 하지만 Boolean 방식으로 사용이 가능하며
		 - `START_DATE <= DATE('2022-10-16') AND END_DATE >= DATE('2022-10-16')`
		 - 해당 식의 비교 결과가 맞는지 (1), 틀린지 (0) 를 확인할 수 있음
		 - And 사용 할 경우 또는 이라는 의미가 되며 식이 하나라도 맞는다면 비교 결과 중 큰 값인 1을 반환
	 - `WHEN MAX(START_DATE <= DATE('2022-10-16') AND END_DATE >= DATE('2022-10-16')) = 1 THEN '대여중' ELSE '대여 가능' END`
		 - Column Start_date 값, Column End_date 값이 조건에 맞으면 결과값이 1이되어 '대여중' 이 출력됨
 - Avg
	 - 평균값을 출력할 때 사용
	 - `Select avg(sal) from emp where job='ANALYST'`
		 - Emp Table 의 job Column 에서 'ANALYST' 에 해당하는 sal Column 행 값 의 `[평균값]`을 출력
 - Count
	 - 결과값을 집계 할 때 사용
	 - `Select count(sal) from emp where job='CLERK'`
			 - Emp Table 의 job Column 에서 'CLERK' 에 해당하는 sal Column 행 값이 `[몇 개 인지]`를 출력
- Stddev
	- 표준 편차 계산에 사용 (통계)
	- 데이터의 변동성(산포도) 평가, 모집단 추론, 품질 관리, 금융 리스크 측정, 학업 성취도 비교(경쟁 정도 파악) 등
