 - Select 절, Where 절, Having 절 등 에서 사용 가능
 - 데이터 타입이 일치하지 않으면 간혹 에러 발생 가능성 있음
 - Coalesce
	 - NULL 을 지정 값으로 치환하는 표현식
	 - 일반적으로 결과값이 Null인 경우 필터링 되어 제외되지만 치환하여 출력 가능
	 - Select
		 - `COALESCE(phone_number, 'Unknown') AS contact_info`
			 - phone_number가 NULL이면 'Unknown'으로 치환
		 - `COALESCE(email_1, email_2, 'Unknown') AS primary_email`
			 - email_1이 NULL이면 email_2를 사용, 둘 다 NULL이면 'Unknown' 사용
	 - Where
		 - `COALESCE(phone_number, 'Unknown') = 'Unknown'`
			 - phone_number가 NULL이면 'Unknown'으로 치환하여 비교
		 - `where coalesce(referee_id, 0)!=2;`
			 - `where referee_id is null or referee_id != 2`
			 - 두번의 비교를 한번 Coalesce 비교로 단축
	 - Having
		 - `COALESCE(AVG(score), 0) < 50`
			 - AVG(score)의 결과가 NULL 일 경우 0으로 치환하여 비교
 - IFNULL (MySQL) / NVL (Oracle) (A,B)
	 - A 가 null 이면 B 로 치환 하는 간단한 치환 밖에 불가능
